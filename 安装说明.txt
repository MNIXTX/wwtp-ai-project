============================================================
  WWTP AI 智慧水务管控平台 — 安装说明
============================================================

【系统要求】
  - Windows 10 / 11 (64 位)
  - Python 3.10 ~ 3.12
  - 内存 >= 8 GB
  - 磁盘 >= 3 GB（含离线包需 5 GB）

============================================================
  一、快速安装（联网环境）
============================================================

  1. 将整个 WWTP_AI_System 文件夹复制到本地磁盘
     （建议 D:\WWTP_AI_System，避免放在桌面或 OneDrive 目录）

  2. 双击 install.bat

  3. 等待安装完成（5~15 分钟），脚本会自动：
     - 检测 Python 环境
     - 创建隔离虚拟环境 venv
     - 安装所有依赖包
     - 验证安装结果

============================================================
  二、离线安装（无外网工控机）
============================================================

  1. 确认 offline_packages 文件夹存在且包含所有 .whl 文件
     （如果缺少，先在联网电脑上运行一次 install.bat 自动下载）

  2. 将整个 WWTP_AI_System 文件夹复制到目标机器

  3. 双击 install.bat（脚本自动检测到无网络，切换离线模式）

  4. Python 不存在时会自动从 offline_packages 中安装

============================================================
  三、启动 / 停止应用（无需终端操作）
============================================================

  启动：双击 start.bat
     - 系统在后台静默启动（无命令行窗口残留）
     - 浏览器自动打开 http://127.0.0.1:8501
     - 提示窗口 3 秒后自动关闭

  停止：双击 stop.bat
     - 关闭后台服务
     - 窗口自动关闭

============================================================
  四、首次使用
============================================================

  1. 准备数据（二选一）：
     A. 将你的 SCADA 历史 CSV 放入 data/ 文件夹
        （需包含 timestamp, flow, inf_cod, eff_cod, inf_nh3,
         eff_nh3, do_meas, DO_reactor, MLSS_reactor 列）
     B. 使用项目中已有的 data/scada_data.csv（模拟数据）

  2. 修改配置（按需）：
     编辑 config.yaml 中的 paths.scada_data_csv 指向你的数据文件
     （首次运行时会自动生成默认 config.yaml）

  3. 进入「模型训练与校准」页面：
     - 展开页面顶部的「数据准备」面板
     - 点击「运行数据管道」完成数据清洗和特征工程
     - 依次点击「LightGBM」「TFT」训练按钮

  4. 训练完成后即可在「手动预测」页面测试模型效果

============================================================
  五、常见问题
============================================================

  Q: 双击 install.bat 闪退？
  A: 右键 install.bat → 以管理员身份运行

  Q: 提示 Permission denied 创建 venv 失败？
  A: ① 确保文件夹不在桌面/OneDrive/网络驱动器/U盘上
     ② 在 Windows 安全中心添加项目文件夹为排除项
     ③ 如果用户名含中文，尝试创建英文用户账户

  Q: 启动后浏览器显示空白？
  A: 等待 10~30 秒，Streamlit 首次加载模型需要时间
     如果超过 1 分钟，查看 logs/ 目录下的日志文件

  Q: TFT 预测值全为 -0.01 或固定在 60 左右？
  A: 模型未训练或训练数据不对。运行「模型训练与校准」页面
     的 TFT 训练按钮重新训练

  Q: 如何更换 SCADA 数据源？
  A: ① 将新 CSV 放入 data/ 目录
     ② 修改 config.yaml 中 paths.scada_data_csv 为新文件名
     ③ 在「模型训练与校准」页面点击「运行数据管道」
     ④ 重新训练模型

  Q: 如何彻底卸载？
  A: 运行 python uninstall.py，按提示选择卸载模式

============================================================
  六、文件夹说明
============================================================

  install.bat        一键安装脚本
  launcher.py        应用启动入口
  config.yaml        全局配置文件（可在线修改）
  requirements.txt   Python 依赖清单
  README.md          详细文档
  安装说明.txt        本文件

  data/              SCADA 数据文件
  models/            训练好的模型权重
  artifacts/         运行时产物（Scaler、特征缓存）
  logs/              运行日志（自动生成）
  ui/                前端界面代码
  offline_packages/  离线安装包（可选）
